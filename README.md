- 🎨 **Modern UI/UX**  
- 🗂️ **Project Section**

- ✉️ **Contact Form** 
- 🌙 **Dark Mode **

- **HTML** – Structure and content
- **CSS** – Styling, layout, and responsiveness
- **JavaScript ** – Interactivity and DOM manipulation
